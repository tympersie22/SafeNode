## ğŸ†˜ Troubleshooting

### Vault Won't Unlock

**Try:**
1. Double-check master password
2. Check if CAPS LOCK is on
3. Try unlocking on web to verify password
4. If forgot password: You'll need to reset (data will be lost)

### Sync Not Working

**Check:**
1. Internet connection (WiFi or mobile data)
2. Sync settings (enabled in Settings)
3. Check for conflicts (tap conflict warning)
4. Try manual sync: Pull down to refresh

### App Crashes

**Try:**
1. Force close and reopen app
2. Restart your device
3. Update to latest version
4. Contact support with crash details

### Biometric Unlock Not Working

**Try:**
1. Check biometric settings in device settings
2. Re-enable biometrics in SafeNode
3. Use master password as fallback
4. Update device OS if outdated
